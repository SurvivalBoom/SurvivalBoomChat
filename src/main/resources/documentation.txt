sbchat.pm - Дозвіл на використання особистих повідомлень.
sbchat.reload - Дозвіл на використання /sbchat reload

########
# ЧАТИ #
########
Ви можете створити безліч різних чатів з унікальними налаштуваннями.
Дозвіл на читання та написання у чат налаштовуються у кожного чату окремо.

У чату є параметри:
enabled - Чи увімкнений чат.
name - Назва цього чату.
moderation - Чи увімкнути модерацію у цьому чаті.
prefix - Префікс для активації цього чату.
write-permission - Дозвіл на написання у цей чат.
view-permission - Дозвіл на перегляд повідомлень у цьому чаті.
format - Формат вигляду чату.
event - Подія яка буде спрацьовувати при написанні у цей чат.

#######################
# ПЕРЕВІРКИ МОДЕРАЦІЇ #
#######################
Плагін SurvivalBoomChat дозволяє створити будь-яку кількість власних перевірок модерації.

У перевірки модерації є параметри:

type - Тип цієї перевірки. Може бути LINK, SWEAR, CAPS.
enabled - Увімкнути?
block - Заблокувати повідомлення із порушенням? Якщо true, повідомлення не буде відправлено. Якщо false, усі порушення з повідомлення будуть прибрані.
admin-message - Повідомлення яке буде надіслано адміністраторам про спрацювання цієї перевірки.
notify-permission - Дозвіл необхідний гравцеві, щоб отримувати сповіщення про спрацювання цієї перевірки.
bypass-permission - Дозвіл необхідний гравцеві, щоб ця перевірка на нього не діяла.
block-message - Повідомлення яке буде надіслано тому на кого спрацювала перевірка якщо block == true.
event - Подія яка буде спрацьовувати при спрацьовуванні цієї перевірки.

LINKS:
 - regex - Regex вирази які будуть шукати шкідкливі посилання.
 - whitelist - Білий список посилань які перевірка буде ігнорувати.
 - replacement - Текст яким буде замінене посилання якщо block == false.
CAPS:
 - length - Мінімальна довжина повідомлення після якої плагін буде перевіряти повідомлення.
 - percent - Скільки відсотків повідомлення повинно бути капсом, щоб правило спрацювало.
SWEAR:
 - replacement - Текст яким буде замінена лайка якщо block == false.
 - file - Назва файла у папці plugins/SurvivalBoomChat/ із лайкою.
